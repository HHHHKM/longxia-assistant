常见问题

有疑问？看这里

🔑 使用 AI 需要"专属口令"是什么意思？去哪里拿？
专属口令就是 AI 服务的 API Key，用来让 ixx.ai 调用 AI 能力。我们推荐用 DeepSeek，注册后充 10 元就能用很久。安装向导里有手把手的图文教程，按步骤操作就行，5分钟搞定。
💰 ixx.ai 完全免费吗？有隐藏收费吗？
ixx.ai 本身永久免费，不收费，不接广告，这是我们的承诺。使用 AI 需要向 AI 服务商（如 DeepSeek、通义千问）充值，但费用很低，10元可以用很久。
🔒 我的数据和聊天记录安全吗？
非常安全。所有配置和聊天记录都只存在你自己的电脑上。ixx.ai 不会把你的任何数据上传到我们的服务器。代码是开源的，任何人都可以验证这一点。
👴 我父母七八十岁了，真的能用吗？
可以。ixx.ai 专门为此设计了老人模式：超大字体、超大按钮、界面极简、操作步骤少。你帮他们做一次初始配置后，他们自己就能聊天和使用自动提醒功能了。
📱 支持微信、飞书等国内渠道吗？
目前推荐使用 Telegram 渠道（设置最简单），飞书渠道也已支持。微信、钉钉渠道正在开发中，即将上线。
🛠️ 我是开发者，可以贡献代码吗？
非常欢迎！项目在 GitHub 上完全开源，MIT 协议。你可以提交 PR、报告 Issue、翻译文档，任何形式的贡献都很有价值。
